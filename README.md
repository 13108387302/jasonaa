# 贪吃蛇网页游戏

一个简单的HTML5贪吃蛇游戏，使用Canvas实现。

## 游戏特点

- 使用方向键或WASD控制蛇的移动
- 空格键暂停/继续游戏
- 响应式设计，适应不同屏幕尺寸
- 简洁美观的界面

## 如何在GitHub Pages上部署

1. 将此项目推送到您的GitHub仓库

   ```bash
   git add .
   git commit -m "初始提交贪吃蛇游戏"
   git push
   ```

2. 在GitHub仓库页面，点击"Settings"标签

3. 在左侧菜单中找到"Pages"选项

4. 在"Source"部分，选择"main"分支和"/(root)"文件夹

5. 点击"Save"按钮

6. 等待几分钟后，您的网站将在`https://[用户名].github.io/[仓库名]`上可用

## 本地运行

只需用浏览器打开`index.html`文件即可在本地运行游戏。

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API
